# desc
keeps nc (netcat) alive after disconnect when option -k is not supported (Debian)

# notes
- Doesn't supoprt mutiple instances
- ncat (from nmap) has the -k option supported, use it if ncat is available. 

# usage
./immortalnc.sh 4444
